# Upgrade all npm packages to latest versions

Install follow package [npm-check-updates](https://npmjs.org/package/npm-check-updates) by


    npm install -g npm-check-updates


 and run 
    npm-check-updates -u


 for upgrade package.json #npm #nodejs #tip
